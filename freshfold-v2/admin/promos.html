How Promo Codes Work

Customers enter their promo code on the Order page or Home page. The discount is automatically calculated and applied to their order total before they confirm via WhatsApp.


To add a new code: Open js/main.js and find the PROMO_CODES object. Add a new entry:

'NEWCODE': { discount: 25, label: '25% off your order!' }

If discount is ≤ 100, it's treated as a percentage. If > 100, it's treated as a flat Rs. amount.
